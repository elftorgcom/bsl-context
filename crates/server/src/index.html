Подключение к клиентам

  • Откройте настройки вашего клиента Antigravity.
  • Добавьте новый MCP-сервер с типом http (Stateless Streamable HTTP).
  • Вставьте сгенерированный JSON-конфиг в ваш конфигурационный файл настроек.
antigravity-config.json
  • Откройте или создайте файл конфигурации Codex config.toml.
  • Глобальный файл находится по пути: ~/.codex/config.toml.
  • Добавьте настройки сервера в секцию [mcp_servers]:
config.toml
  • Откройте файл настроек Claude Desktop claude_desktop_config.json.
  • Этот файл обычно находится по пути:
    %APPDATA%\Claude\claude_desktop_config.json (на Windows)
    или ~/Library/Application Support/Claude/claude_desktop_config.json (на macOS)
  • Добавьте секцию в раздел mcpServers:
claude_desktop_config.json

Доступные инструменты (Tools)

search

Нечёткий поиск по типам, глобальным методам и свойствам 1С с выдачей описания из синтакс-помощника.

info

Получение детальной информации (сигнатура, параметры, описание) по точному имени типа, метода или свойства.

get_members

Получение списка всех членов типа (методы, свойства, значения системных перечислений).

get_member

Получение описания и типов конкретного члена класса/типа 1С.

get_constructors

Получение всех доступных конструкторов типа с их сигнатурами.

get_enum_values

Получение списка возможных значений для системного перечисления.

validate_enum

Валидация значения перечисления на предмет корректности и соответствия стандартам 1С.

validate_method_call

Проверка корректности вызова метода 1С (число переданных аргументов, обязательные параметры).

validate_expression

Глубокий статический анализ BSL-выражения на предмет ошибок типов с выводом confidence-метрик (v0.4.2+).
